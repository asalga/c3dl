C3DL Frustum Culling Test

This pages tests C3DL frustum culling
This test has multiple camera veiws and tells the user if the object is inside or outside the camera view.

To use this test:
1) Select axis or axes in which to apply linear velocity.
2) Click the "Fire!" button.
3) When the object leaves the veiw of the camera the FPS should increase and the text field should say "Outside".
